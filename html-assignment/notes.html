5 html tags I learnt

<a> tag
Creates hyperlinks.
<img> tag
Inserts images.
<table> tag
Creates tables. Use <tr> and <td> to create rows and columns.
<input> tag
Creates spaces for users to enter data.
<div> tag
Groups elements together.
